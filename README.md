# Голос по фактам

[Открыть публичный сайт](https://cozycactus.github.io/golos-po-faktam/)

Русскоязычный civic-tech прототип, который показывает заявления политиков рядом
с официальными текстами законов и поимёнными голосованиями Кнессета.

Первое досье посвящено голосованию `43884` по поправке № 4 к Основному закону:
Судебная власть. Страница отделяет окончательное третье чтение от голосований по
отдельным поправкам, показывает все 70 зарегистрированных результатов и
объясняет границы возможного вывода.

## Источники

- `KNS_PlenumVote` — сведения о голосовании;
- `KNS_PlenumVoteResult` — поимённые результаты;
- официальный опубликованный текст закона в базе Кнессета.

При недоступности внешнего API сайт использует проверенный снимок тех же 70
строк. Отсутствие депутата в поимённой записи никогда не трактуется как голос
«против».

## Локальный запуск

```bash
npm install
npm run dev
```

Проверка перед публикацией:

```bash
npm run build
npm run lint
node --test tests/rendered-html.test.mjs
npm run test:pages
```

`npm run build` собирает серверную версию для Sites. `npm run build:pages`
создаёт полностью статическую версию в `dist-pages/` для GitHub Pages. При
публикации ветки `main` GitHub Actions автоматически проверяет и обновляет сайт.
